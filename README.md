# 欢迎来到老江湖的博客

> 记录进入房间门后的美好

一个基于 [Hexo](https://hexo.io/) 和 [Chic](https://github.com/Mr-X2016/hexo-theme-Chic) 主题构建的个人技术博客。

**作者**: 老江湖、meisijiya

**简介**: 一年前，孤苦伶仃的老江湖，偶然间发现一处房间门，推门而入，从此进入了充满幸福的世界。

## 功能特性

- **归档** - 按时间整理的博客文章列表
- **分类** - 文章分类导航
- **标签** - 文章标签系统
- **任务计时器** - 内置计时工具
- **时间线** - 时光轴展示
- **网站统计** - 访客数据分析
- **照片墙** - 图片画廊
- **本地搜索** - 基于 hexo-generator-searchdb
- **代码复制** - 一键复制代码块功能
- **MathJax** - 数学公式支持
- **评论系统** - 集成评论功能

## 技术栈

- **框架**: Hexo 7.3.0
- **主题**: Chic
- **部署**: GitHub Pages
- **依赖**:
  - hexo-generator-archive
  - hexo-generator-category
  - hexo-generator-index
  - hexo-generator-searchdb
  - hexo-generator-tag
  - hexo-renderer-marked
  - hexo-deployer-git

## 快速开始

```bash
# 安装依赖
npm install

# 本地预览
hexo server

# 生成静态文件
hexo generate

# 部署到 GitHub
hexo deploy
```
