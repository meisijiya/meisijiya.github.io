常见陷阱

  1. 缩进：Python用缩进表示代码块，不是花括号
  2. 动态类型：Python不需要声明变量类型
  3. None判断：用 is None 而不是 == None
  4. 列表复制：list2 = list1 是引用，用 list2 = list1.copy() 复制
  5. 字符串拼接：避免在循环中用 +，用 list + join()

性能优化技巧

  1. 列表推导式：比循环快
  2. 内置函数：sum(), max(), min(), sorted() 比手写快
  3. **适当使用deque**：频繁头部操作时比list快
  4. 避免重复计算：用变量存储len(arr)而不是每次循环都调用
